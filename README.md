Projeto de um formulário de cadastro de usuário utilizando HTML, CSS e JavaScript. O sistema realiza validações dos campos obrigatórios, formato de e-mail, regras de senha, confirmação de senha, idade mínima e CPF, com o objetivo de garantir a integridade dos dados antes do envio do formulário.
